## Thread Fundamentals

**Optimal Length:** 5-7 tweets
**Maximum Recommended:** 12 tweets
**Minimum:** 3 tweets (below this, just write a longer single tweet)

**Structure:**
1. Hook tweet (must grab attention)
2. Body tweets (deliver value)
3. Conclusion tweet (CTA + summary)

## Thread Writing Tips

### Hook Writing (Tweet 1)

**Strong Hooks:**
‚úÖ "I made $10K in 30 days with [method]"
‚úÖ "Everyone is wrong about [topic]. Here's why:"
‚úÖ "I analyzed 100 [things]. Here's what I learned:"
‚úÖ "The 5 [tools/tips/tactics] that [benefit]"

**Weak Hooks:**
‚ùå "A thread about [topic]"
‚ùå "Some thoughts on [topic]"
‚ùå "I want to talk about [topic]"

### Numbering Threads

**Recommended Format:**
```
1/ First point
2/ Second point
3/ Third point
```

**Alternative (if 10+ tweets):**
```
1/12 First point
2/12 Second point
...
12/12 Final point
```

### Pacing

**Vary Tweet Length:**
- Short tweet (50-80 chars)
- Longer tweet (200-250 chars)
- Medium tweet (120-160 chars)
- Short tweet (60-90 chars)

Rhythm keeps readers engaged.

### Hashtags in Threads

**Only use hashtags on:**
- First tweet (1-2 hashtags)
- OR final tweet (1-2 hashtags)

**Never:** Every tweet (looks spammy)

## Thread Publishing Tips

### Timing
- Post first tweet
- Wait 5-10 seconds between subsequent tweets
- Don't rush (gives Twitter time to link them)

### Pinning
- Pin successful threads to your profile
- Drives traffic to valuable content
- Shows new visitors your best work

### Engagement
- Reply to first comment immediately
- Engage with quote tweets
- Retweet the thread next day for second wave
